# Athena AVA Intelligence

Plataforma de automação e inteligência acadêmica para ambientes Moodle/AVA.

## Funcionalidades

- Login automático no Moodle
- Download automatizado de relatórios
- Coleta de acessos dos alunos
- Extração de notas
- Monitoramento acadêmico
- Dashboards educacionais
- Relatórios institucionais
- Analytics para coordenação

## Tecnologias

- Python
- Selenium
- Pandas
- Plotly
- OpenPyXL

## Estrutura

Projeto desenvolvido para automação acadêmica e análise institucional.
